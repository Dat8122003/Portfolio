# Portfolio

Personal portfolio built with React + Vite + TypeScript + Tailwind CSS.

## Develop

```bash
npm install
npm run dev
```

Open <http://localhost:5173/>.

## Build

```bash
npm run build
npm run preview
```

The static site is emitted to `dist/`.

## Configuration

Copy `.env.example` to `.env.local` and fill in the URLs you want to expose
(social, project demo / docs / source). Empty values render a "not
configured" placeholder.
